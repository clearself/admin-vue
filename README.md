# admin-vue (后端配置权限)

English | [简体中文](./README-zh.md)

## Build Setup

```bash
# clone the project
git clone https://github.com/clearself/admin-vue.git

# enter the project directory
cd admin-vue

# install dependency
npm install

# develop
npm run serve
```

This will automatically open http://localhost:9528

## Build

```bash
# build for production environment
npm run build:prod
```
